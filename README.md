# BASIC Auto Transpiled To C (BAT2C)

Very simple program to transpile BASIC code to C and then compile generated C code.

Dependencies:
* gcc
* make

## Usage (WIP)
```
make

./main <file.bas>

gcc <file.bas> -o file

./file
```
